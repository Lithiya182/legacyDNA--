# LegacyDNA Judge Demo Flow

## Duration

3-5 minutes

---

## 1. Introduce the Problem (30 seconds)

Student organizations lose valuable institutional knowledge every year when organizers graduate or leadership changes.

Important lessons, sponsor history, logistics decisions, and planning mistakes are often scattered across documents and eventually forgotten.

LegacyDNA preserves organizational memory and makes it searchable, explainable, and actionable.

---

## 2. Explain Why Memory Matters (30 seconds)

Traditional search finds documents.

Memory systems remember experiences, relationships, outcomes, and patterns across time.

LegacyDNA transforms historical event records into an organizational memory layer powered by Cognee.

---

## 3. Demonstrate Remember() (30 seconds)

Show previously ingested event documents.

Explain that event reports, sponsor records, planning notes, and meeting documents are converted into memory objects stored in Cognee's graph-vector memory system.

---

## 4. Demonstrate Recall() (1 minute)

Ask 2-3 validated demo-safe questions.

Examples:

* Who was the caterer for TechFest 2025?
* Which sponsor paid earliest across all years?
* What was attendance for TechFest 2023, 2024, and 2025?

Show:

* Answer
* Supporting memories
* Source references

---

## 5. Demonstrate Improve() (1 minute)

Open Insights.

Show recurring patterns discovered across multiple years.

Examples:

* Venue booking lessons
* Sponsor retention patterns
* Catering decisions
* Logistics risks

Explain that the system identifies organizational learning from historical memory.

---

## 6. Demonstrate Recommendations (45 seconds)

Open Recommendations.

Show how recommendations are generated using historical evidence rather than generic AI suggestions.

Highlight:

* Recommendation
* Supporting evidence
* Source memories

---

## 7. Why Cognee? (30 seconds)

Cognee provides:

* Memory ingestion
* Knowledge graph construction
* Memory retrieval
* Cross-memory reasoning

LegacyDNA uses memory as the central product capability rather than as a simple chatbot add-on.

---

## 8. Future Roadmap (15 seconds)

Future work includes:

* Forget() lifecycle support
* Memory aging and retirement
* Multi-organization memory spaces
* Long-term institutional knowledge analytics

---

## Closing Statement

LegacyDNA helps organizations preserve institutional memory, learn from past experiences, and make better future decisions using Cognee's memory architecture.
