# LegacyDNA
## What it does
LegacyDNA prevents organizational amnesia by transforming scattered event reports, meeting notes, and sponsor docs into an intelligent, queryable memory graph using Cognee.

## Tech Stack
- Frontend: Next.js
- Backend: Python / FastAPI
- Memory: Cognee
- Database: SQLite

## How to run locally
1. Navigate to the backend folder:
   `cd backend`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the FastAPI server:
   `uvicorn app.main:app --reload`
