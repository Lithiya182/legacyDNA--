# LegacyDNA
## What it does
LegacyDNA prevents organizational amnesia by transforming scattered event reports, meeting notes, and sponsor docs into an intelligent, queryable memory graph using Cognee.

## Tech Stack
- Frontend: Next.js
- Backend: Python / FastAPI
- Memory: Cognee
- Database: SQLite

## How to run locally
1. pip install -r requirements.txt
2. uvicorn main:app --reload
